Design Process

The idea and design for the project appeared while creating a seminar paper on visualizing resumes. There have been several iterations of the design for frontend, backend, and functionality of the system. The client-side design will be improved and the back-end wil change completely throughout the next few months as the Global Resume Standard is developed and brought to life.

Launching the Application

So far, you can launch the application by typing the following lines into your console (relative starting point is the root directory of the server):


# cd src
# go run cv2.go main.go osops.go

In your browser, go to localhost:8080.

Features

Currently, the program features a form where users can submit their data which is then copied into .cv2, .html, .svg files that they can download. The functionality is not yet completed neither do some things work - come back beginning of 2016. This version is solely for developers.
